Задача: "Система управления библиотекой"
Разработайте программу для управления небольшой библиотекой. Программа должна предоставлять следующие возможности:
Требования:
    Создайте базовый класс Book с защищёнными полями:
        title (название книги)
        author (автор)
        year (год издания)
        is_available (доступна ли книга для выдачи)
    В классе Book реализуйте: // абстрактный
        Конструктор для инициализации полей
        Виртуальный метод display_info(), выводящий информацию о книге
        Методы для изменения статуса доступности книги
        Виртуальный метод get_type(), возвращающий тип издания
    Создайте производный класс TextBook, который добавляет: // учебник
        Поле subject (предмет/дисциплина)
        Поле grade_level (класс/курс)
    Создайте производный класс FictionBook, который добавляет: // художественная книга
        Поле genre (жанр)
        Поле age_rating (возрастной рейтинг)
    В функции main() создайте массив указателей на Book и выполните:
        Добавление различных книг (обычных, учебников, художественных)
        Организуйте меню с возможностями:
            Просмотр всех книг в библиотеке
            Поиск книг по автору
            Поиск книг по году издания
            Выдача книги (изменение статуса доступности)
            Возврат книги
            Просмотр только доступных книг
            Просмотр книг определенного типа
    Особые условия:
        Используйте циклы для организации меню
        Применяйте оператор if для проверки условий
        Используйте динамическое приведение типов для идентификации специальных типов книг
        Обеспечьте обработку ошибочных ситуаций
