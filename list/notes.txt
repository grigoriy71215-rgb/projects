В С++ коннструктор по умолчанию (default constructor) - это конструктор, который может быть вызван без аргументов. Он либо не имеет параметров, либо все его параметры имеют значение по умолчанию.

Особености конструктора по умолчанию:
1.Автомотически генерируется  компилятором, если вы не объявили никаких других конструкторов.
2.Инициализирует члены класса их значениями  по умолчанию (для примитивных типов - неопределенное значение, если не используется инициализация по умолчанию).

В с++ типы данных делятся на базовые (значимые) и ссылочные. Они отличаются способом хранения и передачи данных.

1.Базовые (значимые) типы (Value Types)
Хранятся непосредственно  в стеке (если не яввляюстя частью объекта в куче). При передаче в функцию или присваении копируется значение.

Основные категории:

Встроенные примитивные типы:

  int, float, doble, char, bool, etc.

Пользовательские типы:

  struc Point { int x; int y; };
  Point p1; // хранится по значению  

Указатели (Т*) (хотя техически )

Файл .h называется заголовочным файлом, он содержит описание класса, который мы собираемся написать.

Файл .cpp называется файлом реализации, в нем мы создаем реализацию нашего класса, то есть помещаем логику, как методы класса должны работать.

Связаный список построен из объектов, обычно называемых ячейками. Этот класс содержит все данные, которые должны хранится в списке, и ссылку на другую ячейку. Ссылка представляет собой справку или указатель на объект такого же класса. Поле типа "указатель" в ячейке часто называется Next. 

Константность (const corrtctness) - это механизм в с++, который позволяет указывать, что объект или его части не могут быть изменены после создания. Это важная концепция для обеспечия безопастности кода и предотвращения случайных изменений данных.
